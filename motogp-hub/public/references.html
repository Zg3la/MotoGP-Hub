REFERENCES

Sources and technologies used in the development of MotoGP Social Hub

Backend Technologies
Node.js
JavaScript runtime environment. https://nodejs.org
Express.js
Minimalist web framework for Node.js. https://expressjs.com
NeDB (Datastore)
Embedded JavaScript database (MongoDB-like API, file-based). https://github.com/louischatriot/nedb
jsonwebtoken
JSON Web Token implementation for authentication. https://github.com/auth0/node-jsonwebtoken
bcryptjs
Password hashing library. https://github.com/dcodeIO/bcrypt.js
cors
Cross-origin resource sharing middleware for Express. https://github.com/expressjs/cors
Frontend Technologies
HTML5
Semantic markup for the Single Page Application structure. MDN Web Docs
CSS3 (Custom)
Custom responsive design with CSS variables, Flexbox and Grid. No frameworks used. MDN Web Docs
Vanilla JavaScript (ES6+)
Client-side logic, AJAX (Fetch API), DOM manipulation, form validation. MDN Web Docs
Google Fonts – Barlow Condensed & Barlow
Typography. https://fonts.google.com
Hosting & Deployment
Render.com
Recommended free hosting platform for Node.js apps. https://render.com
Railway.app
Alternative PaaS platform for Node.js deployment. https://railway.app
Learning Resources
MDN Web Docs
Comprehensive documentation for web technologies. https://developer.mozilla.org
Express.js Documentation
Official guide for building REST APIs with Express. https://expressjs.com/en/guide/routing.html
JWT.io
Introduction to JSON Web Tokens. https://jwt.io/introduction
MotoGP Official Website
Source for rider information, teams and season data. https://www.motogp.com
Design Inspiration
MotoGP Official Visual Identity
Color schemes, typography style and branding reference from MotoGP media materials.
